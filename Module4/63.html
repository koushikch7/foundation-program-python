When

to

diﬀerent
implementation for one or more inherited method(s),
then the method(s) should be redeﬁned in the child
class this is known as method overriding.
•
A call to method on child class instance will search for
the overridden methods ﬁrst, if not found then the
system will search the method in super class.
•
Any method in child class when wants to access its
parent class instead of overridden method then it can
make use of
super().
The call using super() will search
for the method in parent class instead of child class.
