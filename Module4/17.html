Self is a reference to the current object, i.e the object which invoked the method.
•
Used to create and initialize instance variables of a class i.e. it creates the attribute for the class
•
‘self’ reference must be used as a ﬁrst parameter in all instance methods of a class otherwise the methods are
known as simply “class methods”
•
Moreover, “self” is not a keyword and has no special meaning in Python. We can use any name in that place.
However, it is recommended not to use any name other than “self”
(merely a convention and for
readability)
