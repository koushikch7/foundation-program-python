On completion of this module, the learner should be able to:
  • describe and implement following Object Oriented Concepts:
    • Abstraction
    • Encapsulation
    • Class and Object
    • Inheritance
    • Polymorphism
  • identify Entities, Properties and Behavior in the given requirement description
  • identify and implement relationships among the classes
  • draw UML class diagrams
  • raise and handle built-in and user defined exceptions
  • write simple Unit Tests for a method.
