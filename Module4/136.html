Other techniques used for performing Unit Testing are :

Branch Coverage
: Each branch in control ﬂow must traverse once. For example, an if condition generates
two branches (for true and false respectively) the input parameters should be such that for every such
branch in code is traversed.

Condition coverage
: Input parameters must be such that each condition has to be true at least once and
FALSE at least once. For example (a > 5 and b < 6) the value set for a and b should make this condition
true and false.
