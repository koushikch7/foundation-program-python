How many test cases should be written to test a unit ?
•
For example: if the requirement states that the a password is valid when it is minimum 4 character and maximum
8 character long. There can be inﬁnite number of test cases possible with varying number of characters- 0,1,2,3…
and so on.
•
For all practical reasons, minimal number of test cases are identiﬁed.
•
One such way is to identify boundary values. In this example 4 and 8 are lower and upper boundaries.
•
