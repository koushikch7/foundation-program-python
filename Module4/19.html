The reference returned by a newly created object must be tagged to a variable and that is known as reference
variable
•
Following syntax can be used to create a reference variable for the Customer class and make it point to a
Customer object
•
As there is no need for declaration in Python, custobj will be identiﬁed as reference type automatically during
the runtime.
•
In python, if you end with semicolon (;) it will not display any error. However, ending semicolon is not in the
statement syntax of python.
custobj = Customer()
custobj = Customer();
No ERROR!
