Once a program is written, it should be tested for its correctness, this process is called Testing.
•
In order to test the software, we need to create some test cases.
•
A test case is a set of conditions under which a tester will check whether the system satisﬁes the requirements
and works correctly.
•
The data used for executing a test case is called test data.
•
