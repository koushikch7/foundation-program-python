• Left Outer Join returns the matching rows based on join condition just like Inner Join.
  • In addition it also returns the unmatched rows from the table in left side (table1) assuming NULL for corresponding columns in right side table.
  • Syntax is as follows:
SELECT <<column list>> FROM <<table1>> LEFT OUTER JOIN <<table2>> ON <<join condition>>;
