• It is possible to join more than two tables. Here N-1 join conditions would be required to join N tables.
  • Syntax is as follows:
SELECT <<column list>> FROM <<table1>> INNER JOIN <<table2>> ON <<join condition1>>
INNER JOIN <<table3>> ON <<join condition2>>;
  • This can be represented as:
  • Query for the given scenario can be written as follows:
SELECT s.studentid, s.name, f.amountpaid, d.deptname FROM student s INNER JOIN
feepayment f ON s.studentid = f.studentid INNER JOIN department d ON s.deptno = d.deptno;
