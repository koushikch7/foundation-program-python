Following are the categories of Scalar data types in PL/SQL
  • Numeric
  • Character
  • Boolean
  • Datetime
  • Interval
There are sub types to these data types in PL/SQL.
For eg: Numeric type has got INTEGER, FLOAT and so on, Character has different types like CHAR, VARCHAR2, RAW and so on.
Please refer to http://docs.oracle.com/cd/B28359_01/appdev.111/b28370/datatypes.htm for more details on this.
