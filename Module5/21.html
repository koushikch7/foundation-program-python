• A Sub Query is best defined as a query within a query.
  • Sub Queries can be used in
    • SELECT Clause
    • FROM Clause
    • WHERE Clause
    • HAVING Clause
  • Types of sub queries
    • Independent Sub query – Inner query runs independent of outer query
      • Single Row - No Value or One value as output from inner query
      • Multiple Row – One or More values as output from inner query
      • Multiple Column – Inner query returns multiple columns
    • Correlated Sub query - Inner query refers to a table in the FROM clause of outer query
