• Joining a table with itself.
  • Aliases are created for the same table and join condition is written based on attributes of table aliases.
  • Syntax is as follows:
SELECT <<column list>> FROM <<table1>> <<table1 alias>> INNER JOIN <<table2>> <<table2 alias>> ON <<join condition>> ;
