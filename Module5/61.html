• Oracle creates a context area in memory for processing an SQL statement which contains all information needed for processing
    the statement.
  • A Cursor is a pointer to this context area.
  • Types of Cursor:
    • Implicit Cursor
    • Explicit Cursor (Not discussed in the scope of this course)
