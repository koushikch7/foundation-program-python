• PL/SQL Blocks seen so far were anonymous (unnamed) and they were not re-usable modules.
  • Stored Procedure is a named PL/SQL Block which can be compiled and stored in executable form in the database.
CREATE [OR REPLACE] PROCEDURE <<procedure name>> (<<optional parameter list>>)
IS
<< Declaration section (optional) >>
BEGIN
<< Execution section with PL/SQL statements (mandatory) >>
Exception
<< Exception handling section (optional) >>
END <<procedure name>>;
