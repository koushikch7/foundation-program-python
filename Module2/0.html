On completion of this module, the learner should be able to:
  • Recognize the need for data storage, DBMS and RDBMS
  • Implement Database Development Life Cycle (DDLC)
    • Data Requirement
    • Create ER Model
    • Convert ER Model to Relational Schema
    • Implement Relational Schema using SQL Queries
  • Write basic SQL Queries - using Operators, Built-in functions, Group By Having
