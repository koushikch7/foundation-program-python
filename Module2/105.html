Any column or expression in the SELECT list that is not an aggregate function (AVG,SUM, MIN, MAX,COUNT) must
be in the GROUP BY clause
•
Don’t put group functions in the WHERE clause – use HAVING to restrict groups such as: HAVING AVG(salary)
<1000
•
You can use the GROUP BY clause without using a group function in the SELECT list
•
You cannot use a column alias in the GROUP BY clause
•
The GROUP BY column does not have to be in the SELECT list
•
You can GROUP BY multiple columns, the group function applies to the last column in the list
•
Group functions operate on sets of rows to give
one result per group
.
•
All group functions ignore NULL values
