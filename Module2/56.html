A relation R is said to be in the Third Normal Form (3NF) if and only if :
–
It is in 2NF and
–
There is no transitive dependency which exists between key
attributes and non-key attributes through
another non-key attribute
•
To make a table 3NF compliant, we have to remove all such transitive dependencies by decomposing the relation
