•
A function invoking itself is referred to as Recursion.
•
When a program employs recursion the function invokes itself with a smaller argument.
•
For example: Computing factorial(5) involves computing factorial(4), computing factorial(4) involves computing
factorial(3) and so on.
•
Watch this video to learn how recursion works in Python:
•
A recursive function terminates, if with every recursive call the solution of the problem downsizes and moves
towards a base case.
•
Base/termination case is a condition, where the problem can be solved without further recursion.
•
Recursion can lead to an inﬁnite loop, if the base case is not met.
