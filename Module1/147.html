•
Python has a way to put related code in a ﬁle and use that ﬁle in other Python ﬁles. It is called a
module
.
•
It allows logical organization of code. It can be used to deﬁne variables, functions and classes.
•
Grouping related code into module makes it easier to understand and use.
•
Module is nothing but a Python ﬁle with '.py' extension.
•
Module must be imported before using it's functions in any other module/python file.
•
Modules should have short, all-lowercase names. Underscores can be used in the module name to improve
readability.
•
Syntax to import a module:
import <<
modulename>>
