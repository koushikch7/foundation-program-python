Scope of variables
•
Determines accessibility of a variable at various portions of the program
Diﬀerent types of variables
•
Local variables
•
Variables deﬁned inside the function have local scope
•
Can be accessed only inside the function in which it is deﬁned
•
Global variables
•
Variables deﬁned outside the function have global scope
•
Variables can be accessed throughout the program by all other functions as well
