•
The code(binary) and data of the Python program should be present in some part of main memory (RAM) for
execution (This responsibility lies with the operating system)
•
The main memory allocated for program execution is partitioned into code, stack and heap.
•
All Python objects (values of variables) are stored in heap. Stack contains references pointing to corresponding
objects in heap.
•
Assume a simple Python program:
a = 10
