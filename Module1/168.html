Substitute Function
•
Substitutes all the occurrences of a 'pattern' in the given string with the 'replace pattern'.
•
Returns:
•
Substituted string if 'pattern' is matched
•
Original string if 'pattern' is not matched
•
In this example "low" is a regular expression pattern being searched in the string
"Python is a low level
language" and "high" is the replacement string. As, the pattern "low" is found in the string, it will be replaced by
replacement string "high". Hence the output is "Python is a high level language"
