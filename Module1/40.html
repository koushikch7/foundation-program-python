•
Input() function is used to take keyboard input from user.
•
Reads the input, converts it into a string and returns the same.
•
Example:
name = input("Enter your name: ") # If user enters "John"
print("Hi", name)
# Prints "Hi John“
•
Consider another example:
number = input("Enter a number: ")
result = number + 2
# error
print(result)
This code throws an error at line number 2, as 'number' is of the type string.
•
By default, input() function returns a string. However, input() function can also be used to take user inputs of
diﬀerent data types by using type casting.
•
Type casting converts the input string to a desired data type.
•
In the above example we need to convert the input string 'number' to an integer value. It can be done as
follows:
number = input("Enter a number: ")
# If user enters 2
num = int(number)
# Type casting
result = num + 2
# result will be 2+2=4
print(result)
