In programming, there are two ways in which arguments can be passed to functions –
pass by value
and
pass
by reference
•
Watch this video to understand the diﬀerence between the two approaches:
•
In pass by value, the called function creates a copy of the variable passed to it as an argument. The actual
variable is not aﬀected.
•
In pass by reference, the actual variable is passed to the called function. Hence all the changes made to the
variable inside the function aﬀect its original value.
•
Let us understand the concept of pass by value and pass by reference using an example. Assume, we create a
function change_value() to change the value of a variable.
•
First, trace this pseudo code using pass by value. If a programming language supports pass by value, only the
value of 'a' gets copied into 'num1'. The actual parameter 'a' is not changed inside the change_value() function.
Both the display functions print the same value of 'a' i.e. 10. Hence the desired functionality of updating a value
is not achieved.
function change_value(num1){
num1 = 20
}
a = 10
display(a)
change_value(a)
display(a)
