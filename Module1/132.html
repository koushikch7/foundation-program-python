•
When a Python script raises an exception, it must either handle the exception immediately otherwise it
terminates and quits.
•
Any piece of code which is susceptible to raise an exception should be defended elegantly.
try:
susceptible code
except:
exception handler
•
try…except..
•
If any exception occurs in the 'try' block, 'except' block is executed.
•
Once the 'except' block is complete, the code continues to execute the statements outside the try-except
block.
•
If there is no exception, 'except' block is not executed.
