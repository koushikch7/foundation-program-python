•
else
block can be used to write some statements which we want to execute after try block in case there is no
exception.
•
Either 'except' or 'else' block will be executed.
•
Code in else: block executes if the code in the try: block does not raise an exception.
•
else: block is not executed if try: block raises any exception.
If there is no
exception, else block
is executed
