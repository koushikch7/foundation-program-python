Dictionary can be used to store an unordered collection of key-value pairs.
•
Keys are unique values used to access elements in dictionary.
•
In list , we can only use number(index) to get the data, however in dictionary we can use anything(string,
number, list, etc.) as the key.
•
Unlike lists and tuples, Dictionaries use hashing (an eﬃcient way of accessing elements) instead of indexing.
