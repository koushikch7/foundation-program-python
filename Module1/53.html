An Integrated Development Environment (IDE) can also be used to write and execute programs.
•
"An Integrated Development Environment (IDE) is a software application that provides comprehensive facilities
to computer programmers for software development. An IDE normally consists of a source code editor, build
automation tools and a debugger. Most modern IDEs have intelligent code completion." – Wikipedia
•
Advantages of using an IDE:
•
Less time and eﬀort in writing the code
•
Enforce code quality standards
•
Project Management
•
Easy debugging
•
Various IDEs available for Python are:
•

•

•
PyCharm
•
NetBeans etc.
