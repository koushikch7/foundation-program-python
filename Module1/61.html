Consider the scenario of computing average marks of a student in ABC Training Institute again. We used an
if-else block to print if a student is pass or fail based on his average marks.
•
Let us assume we want to extend the program to print pass/fail status for 60 students of a class.
•
Would you write the same lines of code 60 times?
•
No, we shall use a loop.
•
Loops repeat a block of statements until a speciﬁed condition is true.
