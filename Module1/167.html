Search Function
•
Used to search the ﬁrst occurrence of a pattern in the given string.
•
Returns 'match object' if pattern is found, otherwise returns
none
.
•
We can use group() function of the 'match object' to fetch the matched expression.
In this example “python” is a regular expression pattern being searched in the string “Python is a high level
language”. And 're.I' indicates case insensitive search. Hence matched.group() will return a string “python”.
