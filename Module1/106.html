Now, trace this function using pass by reference. If a programming language supports pass by reference, the
actual parameter is passed to the function change_value() such that num1 = a. The values of a is changed
inside the function, and the display functions print 10 and 20 respectively. Hence the change_value() function is
giving the intended output.
•
However, everything in Python is an object and Python uses a mechanism called as "pass-by-object" or
"pass-by-object reference".
•
If the passed object is of immutable type, the passing acts like pass-by-value . The object reference is passed
to the function parameters but because immutable objects cannot be modiﬁed, a copy of the objects is
created inside the function.
•
If the passed object is of mutable type, then also object reference is passed to the function. As mutable
objects can be changed, the passed objects are updated , hence the argument passing is like
pass-by-reference.
