We can have diﬀerent actions/messages for diﬀerent types of errors that can be thrown by the code.
•
Single try block can have multiple except blocks.
•
except:
block without any type mentioned is default except block.
•
If an error occurs and matching except block is found, matching except block is executed.
•
If an error occurs and matching except block is not found, default except block is executed.
•
If an error occurs and neither matching except block nor default except block are not found, the program
terminates.
